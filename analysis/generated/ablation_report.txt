The analysis uses real SchemaLens methodology variables normalized from IMDb, FIBEN, and LDBC SNB artifacts.

No MongoDB benchmark is rerun.

## Methodological note
This is a simulated ablation over the measured comparison space. It removes component-specific G classes from the measured SchemaLens-selected space using the normalized analytical metadata and activation evidence.
The root-choice ablation is not simulated here because the benchmark artifacts do not include alternative-root MongoDB configurations for all queries. Testing root choice would require materializing and benchmarking additional candidates rooted at non-selected entities.
